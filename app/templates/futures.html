Challenges and Future improvements

Improving Item2Vec via hyperparameter tuning


As mentioned before, Word2Vec is a group of models which are CBOW, Skip-Gram and Hierarchical softmax.
And each model has a lot of parameters to tune.
Gridsearch on those hyperparameters will affect the performance of the model.

Comparison of other advanced models for the same dataset


Many models uses same dataset to build the recommender system. (e.g. SVD++, regularized SVD ..)
To be more practical, Item2Vec should be improved to be the one of those advanced recommender.

Online learning


Even though, there are few need to recompute time to time, It is not a online learning model.

Incorporate with item review data using Natural Language Processing


Movielens dataset provides the tag data and data to link with IMDB dataset.
Hybrid recommender system using Collaborative Filtering and Natural Language processing will improve the recommender.

Not just movie recommendations


Item2Vec can be used other domains as well, as long as there are people and ratings.
